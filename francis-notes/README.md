## 上線（GitHub Pages）
1. github.com 註冊 → New repository（例如 francis-notes）→ Public → Create
2. Add file → Upload files → 把這個資料夾裡的檔案全部拖進去 → Commit
3. Settings → Pages → Source 選 main、資料夾 /(root) → Save
4. 等 1–2 分鐘，網址會是 https://你的帳號.github.io/francis-notes/

## 發新文章
1. 複製 `_template.html` → 改檔名（英文）→ 填內容
2. 開 `index.html`，複製一個 `<li>…</li>` 區塊 → 改連結與標題
3. 上傳兩個檔 → Commit，網站自動更新
